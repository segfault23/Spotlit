# Wants

The following are things that I would like to have added/modified to the frontend

1. Add sorting/filtering by default in adversary creation feature select menu
2. Adversary creation should have a drop down for weapon damage distance (Melee, Very Close, Close, Far, Very Far)

- The dot is hard to type for that section, it should involve a name and the distance drop down, then concatenate them with the dot
- Drop down for damage type on the 3d12 + X phys or mag

3. UI scaling
   - Would like to test on mobile to understand how to modify it.
   - Adversary screen & feature screen seem quite blank on a big screen, not utilizing space
   - Font choice globally. Would like something more fantasy-forward. Font is really small anyway
   - Buttons for hp/fear/stress for PCs are really small, can we make them easier click targets?
     - The +hp and -hp buttons for adversaries feels off. I'm not sure how much I'll use that. Can we instead make the button tracks just larger so they are easier to modify at a glance?
4. Encounter loader sorts oddly, should sort by last saved (if not already)
5. Latency of the profile page. No loading icon or anything to indicate that something is happening. Clicking "Back" and not getting any response for a second or two makes me think nothing is happening.
